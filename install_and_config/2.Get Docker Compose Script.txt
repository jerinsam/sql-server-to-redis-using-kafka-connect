###### Got to Confluent Github page and Download Docker Compose File #######
url : https://github.com/confluentinc/cp-all-in-one/tree/7.5.0-post/cp-all-in-one
Folder : cp-all-in-one

Download Docker Compose Yaml File


# Execute the below command to pull and configure all the impages defined in the docker-compose.yaml file
# this needs to be executed in the same folder where docker-compose.yaml file exists
docker compose up -d

# To stop Container Services 
docker-compose stop

To connect to services in Docker, refer to the following ports.
This may change, check git Readme file for latest updates and also double check docker compose file: 
  
ZooKeeper: 2181
Kafka broker: 9092
Kafka broker JMX: 9101
Confluent Schema Registry: 8081
Kafka Connect: 8083
Confluent Control Center: 9021
ksqlDB: 8088
Confluent REST Proxy: 8082


####### Install Confluent hub #########
## This is used for installing debezium connectors for Kafka Connect
mkdir confluent-hub
cd ./confluent-hub
## Get the latest zip from https://docs.confluent.io/platform/7.4/connect/confluent-hub/client.html
wget https://client.hub.confluent.io/confluent-hub-client-latest.tar.gz
tar -zxvf confluent-hub-client-latest.tar.gz
